# Vinit Darade Portfolio

This is a dynamic, aesthetic portfolio website built with React, Vite, Framer Motion, and React Router.

## Features
- **Multi-Page Layout**: Sequential flow from Education to Projects.
- **Easy Customization**: Data separated in `src/data/resume.js` and `src/data/projects.js`.
- **Modern UI**: Dark theme, glassmorphism effects, and smooth page transitions.
- **Resume Integation**: Resume download functionality.

## Getting Started

1.  **Install Dependencies**
    ```bash
    npm install
    ```

2.  **Run Locally**
    ```bash
    npm run dev
    ```
    Open the link shown in the terminal (usually `http://localhost:5173`).

3.  **Build for Production**
    ```bash
    npm run build
    ```

## Project Structure
- `src/components/Navigation.jsx`: Top navigation entry.
- `src/pages/`: Individual page components (Education, Skills, Experience, Projects).
- `src/data/resume.js`: **EDIT THIS** to put your actual details.
- `src/data/projects.js`: Edit this for project links.
